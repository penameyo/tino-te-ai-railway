# TINO-TE.ai BETA 설명서

## 소개
TINO-TE.ai는 학생들이 강의 내용을 더 효과적으로 학습하고 정리할 수 있도록 도와주는 AI 기반 노트 생성 도구입니다. 오디오 파일이나 문서를 업로드하면 AI가 자동으로 내용을 분석하고 요약하여 학습 노트를 생성합니다.

## 🎯 TINO-TE.ai BETA 프로젝트 현황 정리

### ✅ **구현 완료된 기능들**

#### **1. 사용자 인증 시스템**
- 학번과 이름을 통한 간편 로그인
- API 키 기반 인증 시스템
- 사용자 정보 조회 API (`/api/v1/users/me`)
- 자동 사용자 생성 (첫 로그인 시)

#### **2. 오디오 파일 처리**
- **실시간 녹음**: 브라우저에서 직접 음성 녹음 (MediaRecorder API 사용)
- **파일 업로드**: MP3, WAV, M4A 등 오디오 파일 업로드 (드래그 앤 드롭 지원)
- **Whisper API 연동**: 음성을 텍스트로 변환 (교육 콘텐츠 특화 프롬프트)
- **DeepSeek R1 API 연동**: 전사된 텍스트를 구조화된 학습 노트로 변환
- **크레딧 시스템**: 오디오 처리 시 10 크레딧 차감
- **크레딧 확인 모달**: 처리 전 사용량 확인 및 동의

#### **3. 문서 파일 처리**
- **지원 형식**: PDF, DOCX, DOC, TXT 파일
- **텍스트 추출**: 문서에서 텍스트 자동 추출
- **AI 노트 생성**: DeepSeek R1으로 핵심 내용 요약 및 구조화
- **크레딧 시스템**: 문서 처리 시 5 크레딧 차감
- **파일 크기 제한**: 10MB 이하 파일 처리

#### **4. 노트 관리 시스템**
- **노트 목록**: 생성된 모든 노트 조회 (최신순 정렬)
- **노트 상세 보기**: 제목, 요약, 원본 텍스트/전사 내용을 모달로 표시
- **노트 삭제**: 개별 노트 삭제 기능 (드롭다운 메뉴)
- **노트 타입 구분**: 오디오/문서 노트 아이콘 및 색상으로 구분
- **노트 생성 후 즉시 보기**: 생성 완료 시 자동으로 상세 내용 표시

#### **5. 크레딧 시스템**
- **일일 크레딧**: 사용자당 하루 10 크레딧 제공
- **자동 리셋**: 매일 자정 크레딧 자동 초기화 (Cron 스케줄러)
- **크레딧 확인 모달**: 처리 전 크레딧 사용량 확인
- **실시간 크레딧 표시**: 사이드바에서 현재 크레딧 확인 가능

#### **6. 관리자 기능**
- **관리자 페이지**: `/admin` 경로로 접근 (비밀번호: admin123)

- **크레딧 관리**: 모든 사용자 크레딧 일괄 리셋
- **API 키 인증**: 관리자 전용 API 보안
- **사용자 통계**: 사용자별 노트 생성 현황 확인

#### **7. UI/UX 기능**
- **다크 모드**: 시스템 설정에 따른 자동 테마 적용

## 기술 스택

### **프론트엔드**
- **Next.js 14**: React 기반 풀스택 프레임워크
- **TypeScript**: 타입 안전성 보장
- **Tailwind CSS**: 유틸리티 기반 CSS 프레임워크

### **백엔드**
- **FastAPI**: 고성능 Python 웹 프레임워크
- **SQLAlchemy**: Python ORM
- **PostgreSQL**: 관계형 데이터베이스
- **Docker**: 컨테이너화
- **APScheduler**: 백그라운드 작업 스케줄링

### **AI 서비스**
- **OpenAI Whisper API**: 음성-텍스트 변환 (교육 콘텐츠 특화)
- **DeepSeek R1 API**: 텍스트 분석 및 구조화된 노트 생성

### **인프라**
- **Docker Compose**: 개발 환경 구성
- **Cron**: 자동화된 작업 스케줄링

## 설치 및 실행 방법

### 사전 요구사항
- Node.js 18 이상
- Python 3.9 이상
- Docker 및 Docker Compose
- OpenAI API 키
- DeepSeek API 키

### 백엔드 설정
1. 환경 설정
   ```bash
   cd backend
   python -m venv .venv
   source .venv/bin/activate  # Windows: .venv\Scripts\activate
   pip install -r requirements.txt
   ```

2. 환경 변수 설정
   `.env` 파일을 backend 디렉토리에 생성하고 다음 내용을 추가합니다:
   ```
   DATABASE_URL=postgresql://postgres:postgres@localhost:5433/tinoteai
   OPENAI_API_KEY=your_openai_api_key
   DEEPSEEK_API_KEY=your_deepseek_api_key
   ```

3. 데이터베이스 실행
   ```bash
   docker-compose up -d
   ```

4. 백엔드 서버 실행
   ```bash
   cd backend
   uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
   ```

### 프론트엔드 설정
1. 환경 설정
   ```bash
   cd frontend
   npm install
   ```

2. 환경 변수 설정
   `.env.local` 파일을 frontend 디렉토리에 생성하고 다음 내용을 추가합니다:
   ```
   NEXT_PUBLIC_API_URL=http://localhost:8000
   ```

3. 프론트엔드 서버 실행
   ```bash
   npm run dev
   ```

## 사용 방법

### 로그인
1. 웹 브라우저에서 `http://localhost:3000`으로 접속합니다.
2. 우측 상단의 로그인 버튼을 클릭합니다.
3. 학번과 이름을 입력하여 로그인합니다.
   - 예시: 학번 `2023148039`, 이름 `박종호`
   - 첫 로그인 시 자동으로 계정이 생성됩니다.

### 노트 생성

#### 오디오 노트 생성
1. 대시보드에서 **"Record or upload audio"** 카드를 클릭합니다.
2. 두 가지 방법 중 선택:
   - **실시간 녹음**: "Record audio live" 버튼으로 직접 녹음
   - **파일 업로드**: 오디오 파일을 드래그 앤 드롭 또는 클릭하여 선택
3. 크레딧 사용 확인 후 처리를 시작합니다. (10 크레딧 사용)
4. AI가 음성을 텍스트로 변환하고 학습 노트를 생성합니다.
5. 생성된 노트가 자동으로 표시됩니다.

#### 문서 노트 생성
1. 대시보드에서 **"Document upload"** 카드를 클릭합니다.
2. PDF, DOCX, DOC, TXT 파일을 드래그 앤 드롭 또는 클릭하여 선택합니다.
3. 크레딧 사용 확인 후 처리를 시작합니다. (5 크레딧 사용)
4. AI가 문서 내용을 분석하고 학습 노트를 생성합니다.
5. 생성된 노트가 자동으로 표시됩니다.

### 노트 관리
1. 대시보드에서 생성된 모든 노트 목록을 확인할 수 있습니다.
2. 노트를 클릭하면 상세 내용을 볼 수 있습니다:
   - **제목**: AI가 생성한 노트 제목
   - **요약**: 핵심 내용 요약
   - **원본 텍스트/전사 내용**: 전체 텍스트 내용
3. 노트 우측의 메뉴(⋯)를 통해 다음 작업이 가능합니다:
   - **View Details**: 상세 내용 보기
   - **Delete**: 노트 삭제

### 크레딧 시스템
- 각 사용자는 하루에 10개의 크레딧을 받습니다.
- **오디오 파일 처리**: 10 크레딧 사용
- **문서 파일 처리**: 5 크레딧 사용
- 크레딧은 매일 자정에 자동으로 리셋됩니다.
- 사이드바에서 현재 크레딧을 확인할 수 있습니다.

## 주의사항
- 이 프로젝트는 베타 버전으로, 일부 기능이 완전하지 않을 수 있습니다.
- 대용량 파일 처리 시 시간이 오래 걸릴 수 있습니다.
- 일일 사용량 제한이 있으므로 중요한 파일부터 처리하는 것을 권장합니다.
- 오디오 파일은 명확한 음성 품질이 좋을수록 더 정확한 결과를 얻을 수 있습니다.
- 문서 파일은 텍스트가 명확하게 추출 가능한 형식이어야 합니다.
